# Changelog

This file documents notable changes to **Cloud‑Compliance‑Engine‑ISO27001**.  It
adheres to the [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) format
and [Semantic Versioning](https://semver.org/).

## [0.1.0] – 2026‑03‑01

### Added

* **Report schema templates:** Added `templates/report_schema.csv` and
  `templates/report_schema.json` defining the columns and structure for ISO 27001
  control mapping, evidence fields, status, owner and notes.
* **Control coverage matrix documentation:** Added
  `docs/control_coverage_matrix.md` describing how ISO 27001 controls map to
  cloud services and compliance checks.
* **Sample sanitized report:** Introduced `samples/sample_report.md` to
  illustrate the output format of a compliance assessment without exposing
  sensitive customer data.
* **Initial best‑practice files:** Added MIT license, security policy,
  code of conduct and contributing guidelines to align with open‑source
  standards.
* **Initial release:** Tagged the first version (`v0.1.0`) to signal the
  project’s readiness for collaborators.
