# Cloud Compliance Engine ISO27001 - Case Study

## Problem

Compliance evidence is often stored manually and inconsistently, making it difficult to review control status and prepare clear summaries.

## Solution

This project demonstrates a lightweight compliance automation baseline that:

- loads structured control metadata,
- loads sample evidence records,
- validates status values,
- aggregates evidence outcomes,
- and generates a Markdown report.

## Outcome

The project provides a repeatable ISO27001-oriented reporting example that can be expanded into broader GRC and cloud compliance workflows.

## Portfolio value

Relevant for cybersecurity GRC, cloud compliance, ISO27001 readiness and security evidence automation roles.
