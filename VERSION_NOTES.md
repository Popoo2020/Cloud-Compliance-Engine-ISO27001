# Version Notes

## v0.1.0 - Portfolio Readiness Release

### Purpose

Initial portfolio-grade release for the Cloud-Compliance-Engine-ISO27001 project.

### Highlights

- Structured control templates.
- Sample evidence records.
- Markdown report generation.
- Status validation.
- Unit tests and CI.
- Bandit review and dependency review.
- CodeQL analysis.
- Architecture documentation.
- Threat model.
- Demo output documentation.
- Quality and security checklist.
- Recruiter summary.

### Known limitations

- Portfolio baseline, not a certification system.
- Sample evidence only.
- No persistent evidence store.
- No reviewer sign-off workflow.
- Markdown-only reporting.

### Future roadmap

- Add JSON schema validation.
- Add evidence freshness tracking.
- Add exception expiry fields.
- Add PDF/HTML output.
- Expand control mapping examples.
